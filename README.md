# Student Event Feedback Analysis

## Project Overview
This project analyzes student event feedback using data analysis and basic Natural Language Processing (NLP) techniques.  
The goal is to understand student sentiment, visualize feedback patterns, and derive actionable insights to improve future events.

The project demonstrates:
- Data cleaning and preparation using pandas
- Sentiment analysis using TextBlob
- Data visualization with matplotlib and seaborn
- Interpretation of feedback for decision-making

---

- Common themes include event quality, engagement, and organization
- Negative feedback highlights pacing and management issues
